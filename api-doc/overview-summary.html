Overview

Ceylan-Oceanic gatehrs Erlang-based facilities for the support of the Enocean building automation system. It is a free software lightweight library implemented in Erlang.

This website presents the documentation of the API (Application Programming Interface) of Ceylan-Oceanic version 0.0.1 (as of Wednesday, September 7, 2022).

Ceylan-Oceanic relies on the Myriad layer, a generic Erlang toolbox.
